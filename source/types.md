# Types

Types in Pokétop RPG work exactly the same as in the (Gen I, for now...) games. Pokémon and moves can have the following types:

* Bug
* Dragon
* Electric
* Fighting
* Fire
* Flying
* Ghost
* Grass
* Ground
* Ice
* Normal
* Poison
* Psychic
* Rock
* Water
